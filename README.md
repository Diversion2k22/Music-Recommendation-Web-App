### Getting Started ✈️ <br>

1. git clone <repo link>
2. cd server and npm install
3. cd client and npm install
4. In server run nodemon index
5. In client run npm start
  
  LET THE MUSIC SPEAK 🎵 👯‍♀️  <br>
